Simple Graphics Demo Using Java Graphics 2D

///////////////////////////////////////////////////////////////////////////////
In order to successfully open and run the program,
your computer must be installed with Java. If your
computer doesn't have a Java installed, please follow
this site: 
  https://www.java.com/en/download/manual.jsp
and choose the right program compatible to your computer.

///////////////////////////////////////////////////////////////////////////////
If you have successfully install Java on your computer,
please follow this instruction:
1. Find my repositories on GitHub and go to AALacanaria/JavaGraphicsDemo
2. Open GraphicsDemoo
3. Open dist
4. Open GraphicsDemoo.jar
5. Fill in the required values
6. You have successfully created your own color of Invoker
